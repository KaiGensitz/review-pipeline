# ROLE

# INSTRUCTIONS
Task description: Read the full-text evidence under the DATA section and the ELIGIBILITY CRITERIA. Conduct a step-by-step logical deliberation. Prioritize explicit PDF evidence over assumptions.
Method-aware policy: If one or more chunks are labeled section method, treat method evidence as the primary source for intervention/outcome judgments (smartphone, artificial intelligence, physical activity) and use other sections only as supporting context.
Intervention-first policy: Papers that only perform measurement, monitoring, characterization, classification, or algorithm benchmarking without an active behavior-change intervention delivered to participants must be excluded.
PA-centrality policy: Exclude when PA is only measured/secondary and not the intervention target. PA remains central when the intervention adapts timing/content/goals/recommendations specifically to improve PA outcomes (for example steps, MVPA, activity minutes, sedentary breaks).
AI-control policy: Count AI as present only when methods evidence shows a learning/adaptive algorithm controlling participant-facing intervention decisions (timing, content, goal, recommendation). Deployed adaptive AI personalization from participant data is acceptable even with brief model detail. Rule-based logic, hard-coded heuristics, static if-then systems, and case-based retrieval without learning updates are non-AI unless the paper explicitly states model/policy updating from participant data during deployment. Proxy wording alone is not sufficient.
Smartphone evidence policy: For smartphone criterion, treat explicit smartphone terms OR clearly deployed participant-facing mobile-app/mHealth/chatbot delivery on personal mobile phones as qualifying smartphone technology, even if the literal word smartphone is absent. Exclude only when evidence is explicitly desktop-only, web-only without mobile deployment, wearables-only without an app, or otherwise non-smartphone.
Protocol-adjudication policy: For protocol/framework/design papers, include only when participant-level intervention exposure and intervention-control mechanism are explicitly defined. If the current paper explicitly reports participant-level intervention allocation/delivery and outcomes, do not classify it as protocol-only.
Publication-type conflict policy: If evidence is mixed (for example dissertation-like context but empirical participant-level intervention methods/results), do not auto-exclude as wrong_publication_type unless non-empirical status is explicit.
Evidence-contamination policy: Bibliography/reference lists, appendices, and unrelated inserted fragments cannot be the sole basis for exclusion. Required triad evidence must come from methods/results or directly linked intervention descriptions in the current paper. If prioritized method/results evidence remains contradictory, use insufficient_context=true.

Do not use bibliography/reference lists, appendix boilerplate, or unrelated inserted text as the sole basis for any definitive exclusion flag. 5) FINAL DECISION: If any exclusion is explicitly met, return false. If all core concepts are present, return true.
